Tags HTML

Voltar

As tags HTML são elementos utilizados para marcar e estruturar o conteúdo de uma página web. Elas fornecem instruções sobre como o conteúdo deve ser exibido no navegador. As tags podem ser de vários tipos, como <p> para parágrafos, <h2> a <h6> para títulos, <a> para links, entre muitas outras.

Atributos de Tags

Voltar

Os atributos são valores adicionados às tags HTML para fornecer informações extras sobre o elemento. Eles podem modificar o comportamento ou a aparência do elemento. Por exemplo, o atributo href em uma tag <a> especifica o link de destino, e o atributo src em uma tag <img> define a origem da imagem.

Textos

Voltar

Os textos são parte essencial do conteúdo de uma página web. Eles podem variar de parágrafos a títulos, sendo marcados por tags específicas. Além disso, é possível estilizar e formatar o texto utilizando tags como <strong> para texto em negrito e <em> para texto enfatizado.

Listas Ordenadas e Não Ordenadas

Voltar

As listas ordenadas e não ordenadas são formas de organizar itens. As listas ordenadas, marcadas por <ol>, têm números ou letras que indicam a ordem dos itens, enquanto as não ordenadas, marcadas por <ul>, utilizam marcadores, como pontos, para mostrar os itens sem uma ordem específica.

Inspetor de Elementos

Voltar

O inspetor de elementos é uma ferramenta dos navegadores que permite visualizar e editar o HTML e o CSS de uma página, facilitando o desenvolvimento e a depuração. Com o inspetor de elementos, é possível inspecionar a estrutura da página, modificar o código HTML e CSS em tempo real e depurar possíveis problemas de layout e estilo.
